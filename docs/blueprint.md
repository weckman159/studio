# **App Name**: AutoSphere

## Core Features:

- Feed of Posts: Display a feed of posts, sorted by date and likes.
- Car of the Day: Showcase a featured 'Car of the Day' with its photo, brand, model, and owner.
- Garage Management: Enable users to manage their car collection, add new cars, and edit details (brand, model, year, photos).
- Car Profile: Dedicated pages for each car with photos, specs, post history, likes, comments, and 'Car of the Day' badge.
- Post Creation: Allow users to create, edit, and delete blog posts with text, photos, and videos; tag them with 'repair', 'tuning', 'travel'.
- User Profile: Display user profiles with photo, name, description, car list, post statistics, and 'Car of the Day' wins.
- Authentication: User authentication through login/registration, Google OAuth, and email/password.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to reflect reliability and technical prowess.
- Background color: Very light grey (#F0F2F5), nearly white.
- Accent color: Soft lavender (#C5CAE9) to suggest subtle energy and creative problem-solving.
- Body and headline font: 'Inter', sans-serif, providing a clean and modern look, for both headlines and body.
- Use modern, line-style icons for navigation and actions.
- A clean, grid-based layout to present information clearly.
- Subtle transitions for loading content and navigating pages.
